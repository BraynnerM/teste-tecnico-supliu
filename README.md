## Visão geral

### O desafio

Os usuários devem ser capazes de:

- Ver lista de álbuns e faixas
- Pesquisar álbuns por nome
- Adicionar um novo álbum
- Adicionar uma nova faixa em um álbum
- Excluir uma faixa
- Excluir um álbum
